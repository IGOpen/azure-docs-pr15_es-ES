Sus máquinas virtuales de IaaS (VM) y PaaS las instancias de función en una red virtual reciben automáticamente una dirección IP privada de un rango que especifique, basado en la subred están conectados a. Esta dirección se conserva por las máquinas virtuales y las instancias de función, hasta que se retiren. Retirar una instancia VM o función mediante la detención de PowerShell, CLI Azure o el portal de Azure. En estos casos, una vez que la instancia VM o función se inicia de nuevo, recibirá una dirección IP disponible desde la infraestructura de Azure, que puede no ser el mismo que tenía anteriormente. Si cierra la instancia de máquina virtual o rol desde el sistema operativo invitado, conserva la dirección IP que tenía.  

En algunos casos, desea una instancia VM o función tiene una dirección IP estática, por ejemplo, si su máquina virtual se va a ejecutar DNS o un controlador de dominio. Para hacerlo, puede establecer una dirección IP privada estática.
