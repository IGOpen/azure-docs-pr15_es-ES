# <a name="operations-management-suite-oms-sdk"></a>Serie de administración de operaciones (OMS) SDK
[Conjunto de aplicaciones de administración de operaciones (OMS)](../operations-management-suite/operations-management-suite-overview.md) es basado en la nube administración solución de Microsoft que le ayuda a administrar y proteger su local y la infraestructura de nube.  Este artículo enumera la documentación y otros recursos disponibles para obtener acceso a servicios OMS fuera de los portales OMS y Azure.  Esto incluye API de REST de varios motores de secuencias de comandos como PowerShell e interfaces de programación de access.
