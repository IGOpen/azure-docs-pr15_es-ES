#<a name="sdk-content"></a>Contenido SDK

Este documento se enumera y describe el contenido del archivo SDK.

##<a name="engagement-sdk"></a>Compromiso SDK

`EngagementSDK`

> Los encabezados de contenedor de compromiso SDK de núcleo, biblioteca y una clase adicional que se proporciona como un código fuente para habilitar o deshabilitar la colección de IDFA.

##<a name="engagement-reach-sdk"></a>Compromiso alcance SDK

`EngagementReach`

> Compromiso alcance SDK que contiene encabezados, biblioteca y recursos asociados.

##<a name="additional-files"></a>Archivos adicionales

`doc/index.html`(en `doc` carpeta)

> Compromiso SDK y la documentación del alcance SDK objetivo-C.

`src`carpeta

> Controladores de la vista de contratación y código de origen del alcance SDK (para usuarios avanzados que desean adaptar las clases de contratación a sus necesidades).
