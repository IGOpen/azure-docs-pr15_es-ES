# <a name="azure-storage-client-tools"></a>Herramientas de cliente de almacenamiento de Azure

Los usuarios de almacenamiento de Azure con frecuencia desean que puedan ver o interactuar con sus datos mediante una herramienta de cliente de almacenamiento de Azure. En las tablas siguientes, se enumeran una serie de herramientas que le permiten hacer esto. Incluimos una "X" en cada bloque si proporciona la capacidad de bien enumerar o acceder a la extracción de datos. La tabla también muestra si las herramientas es gratuita o no. "Prueba" indica que hay una prueba gratuita, pero el producto completo no es gratuito. "Y" o "N" indica que hay una versión disponible de forma gratuita mientras está disponible para su compra una versión diferente.

Solo proporcionamos una instantánea de las herramientas de cliente de almacenamiento de Azure disponibles. Estas herramientas pueden continuar evolucionando y creciendo de funcionalidad. Si hay actualizaciones o correcciones, por favor, deja un comentario para comunicarnos. El mismo es true que si sabe de herramientas que deben ser aquí - podríamos felices agregarlos.
